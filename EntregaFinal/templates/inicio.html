Lo primero es puedes usar todas la funciones de python para representa alguna funciones que quieres evaluar.

  • como escribir funciones de forma correcta.

  • Funcion solo para el metodo: log(sin(x)**2 + 1) - 1/2
  • Error: log(sin**2(x) + 1) - 1/2
  • ex-x
  • Funcion para el metodo y graficadora: exp(x)-x
